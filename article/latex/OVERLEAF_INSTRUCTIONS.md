# Overleaf Instructions

1. Upload `latex.zip` to Overleaf.
2. Set the main document to `main.tex`.
3. Set the compiler to **XeLaTeX**.
4. Recompile.

The archive should contain these top-level entries:

- `main.tex`
- `references.bib`
- `figures/`
- `tables/`
- `README.md`
- `OVERLEAF_INSTRUCTIONS.md`

If bibliography entries do not appear on the first compile, run the usual sequence by recompiling after BibTeX has executed, or use Overleaf's recompile button twice.
