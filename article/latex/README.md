# LaTeX Paper Draft

This folder contains a LaTeX version of the refocused paper:

`From Legal Provisions to Machine-Readable Controls: A Transformation Framework for China AIGC Governance`

This LaTeX package is retained as a superseded prototype-paper artifact. The current main paper draft is maintained in `paper/main.tex` and centers on legal-clause-to-control transformation rather than a full compliance assessment product. The older synthetic request pipeline is treated as supplementary proof of concept.

Recommended compile path:

```bash
latexmk -pdf main.tex
```

This is a research paper draft. It does not provide legal advice, certify compliance, or claim regulatory endorsement.
